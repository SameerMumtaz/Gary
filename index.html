🛠 Adding a New Character
Each player gets their own HTML file. Five steps, no coding required.
  • 1
    Duplicate gary-wicksworth.html In your GitHub repo, copy the file and rename it — e.g. petra-stormhand.html. This is now that player's character file.
  • 2
    Find the CHARACTER_KEY in the file Search for CHARACTER_KEY near the top of the script block. Change it to match the new filename:
    const CHARACTER_KEY = 'petra-stormhand';
    This ensures notes save separately for each character on the same device.
  • 3
    Replace the character content Search through the HTML for the player sheet section and update: name, race/class, alignment, backstory, stats, spells, traits, appearance, want, fear. The structure is already built — just swap the text.
  • 4
    Add their card to this dashboard Open index.html and find an empty character slot. Replace it with:
    <a class="char-card" href="petra-stormhand.html" data-emoji="🔥"> <div class="card-race">Human · Paladin · Lvl 1</div> <div class="card-name">Petra Stormhand</div> <div class="card-player">played by [Name]</div> <div class="card-tagline">One line about who they are.</div> <span class="card-cta">Open Sheet →</span> </a>
  • 5
    Push to GitHub and share the link Commit both files. The new character sheet is live at yourusername.github.io/repo-name/petra-stormhand.html and linked from the dashboard automatically.
